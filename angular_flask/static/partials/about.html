About The Blog

The Blog is the place where can share your thoughts and ideas with others as well as give and receive valuable feedback. It is easy to get started. Create an account, log in and from the user menu in the top right corner go to New post and you're all set! Additionally you can customize the default cover image for your posts to make them stand out and better convey your thoughts.

Found something that you enjoyed in particular? Favorite post to give its author a positive feedback or leave a comment below!

If writing is not your thing you can simply browse the latest posts from Home page or see all of them at one by going to Posts.

Enjoy your stay!

This site is built using three core technologies

Angular.js 1.x

Angular is one of the most popular JavaScript frameworks. It powers The Blog's front-end allowing it to display pages without making unnecessary server requests.

Angular Material

Angular Material is the reference implementation of Google's Material Design Specification. It provides a set of reusable, well-tested, and accessible UI components based on Material Design.

Flask

Flask is a lightweight microframework written in Python. It is perfect for constructing APIs and interacting with databases, which is what it is mainly used here for.
